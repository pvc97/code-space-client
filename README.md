## Flutter Intl

Extension for VS Code: https://marketplace.visualstudio.com/items?itemName=localizely.flutter-intl
Initialize Intl: CRTL + SHIFT + P -> Flutter Intl: Initialize
Add new locale: CRTL + SHIFT + P -> Flutter Intl: Add locale
